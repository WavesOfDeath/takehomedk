# TakeHomeDK

English Denmark salary, tax, visa threshold, researcher-tax and relocation calculator prototype for expats evaluating Danish job offers.

## Run locally

```bash
npm install
npm run dev
```

Open the printed Vite URL. The prototype is a front-end-only MVP with transparent assumptions and source links.

## Current calculator scope

- 2026 salary after tax estimate using AM-bidrag and published 2026 bracket rates.
- Municipality tax presets for common expat locations.
- Employer/employee pension and annual bonus inputs.
- Pay Limit Scheme threshold screen at DKK 552,000/year.
- Researcher/key employee salary signal at DKK 65,400/month.
- Copenhagen-style living-cost leftover calculator.

This is not tax, legal, financial or immigration advice.
